What is it?

"Who's Free?" is a super simple tool.

You just need to paste a text message to find out if someone is currently sleeping, busy, or free, even if you're in different time zones.

It's not precise tracking, nor is it a schedule manager.

Its sole purpose is:

To save you from awkwardly bothering someone.

Why did I create this?

I often chat with online friends and acquaintances in different countries.

Every time I want to reach someone, I have to mentally convert time zones and guess if they're sleeping.

This tool is designed to "let the computer do the work."
